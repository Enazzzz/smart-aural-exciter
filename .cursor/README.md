# .cursor

Project-local Cursor configuration directory.

This directory is intentionally minimal. Add subfolders as needed:

- `rules/`     project rules (`*.mdc` files Cursor reads automatically)
- `commands/`  project-scoped slash commands
- `snippets/`  shared snippets / scaffolding

No files are created here by default; populate manually as conventions emerge.
Cursor docs: <https://docs.cursor.com>
